HTML Paragraphs

HTML Paragraphs

Dùng thẻ <p> để tạo ra 1 đoạn văn bản

Example:
<p>This is a paragraph. </p>
<p>This is another paragraph.</p>

HTML Display

Thẻ <p> khi hiển thị trên page sẽ hiện thị theo 1 đoạn văn, kể cả khi code có xuống dòng hay khoảng trắng thì khi save lại sẽ thành 1 đoạn.

HTML Horizontal Rules

Dùng thẻ <hr> để kẻ 1 đường thẳng ngang page.

HTML Line Breaks

Để xuống dòng thì ta dùng thẻ <br> ở cuối câu mà ta muốn xuống dòng.

HTML <pre> Element

Khi mà muốn viết thế nào hiện thị như vậy thì ta dùng thẻ <pre>
